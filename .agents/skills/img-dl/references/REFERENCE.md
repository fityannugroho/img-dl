## File formats for bulk download

- Text files (`.txt`) - Each line contains a URL [[Example](urls.txt)]
- JSON files (`.json`):
  - Array of URL strings [[Example](urls.json)]
  - Array of URL objects [[Example](urls-per-items.json)]
- CSV files (`.csv`):
  - With header [[Example](urls-with-header.csv)]
  - Without header [[Example](urls-without-header.csv)]
